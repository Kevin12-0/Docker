¿Qué es Docker?

Docker es un proyecto / plataforma de software libre y de cogido abierto, que se usa principalmente para pode automatizar la implementación de aplicaciones como los contenedores portátiles y autosuficientes, ademas de que se puedan ejecutar dichas aplicaciones de una forma local o desde la nube. Ademas de que Docker igual es una empresa que se a encargado de dar un impulso a la tecnología, en colaboración con sistemas como lo es Linux y Windows.

Actualmente esta trabajando de la mano con lo que es AWS para poder ayudar a los desarrolladores a una mejor entrega de aplicaciones que tengan como base el almacenamiento en la nube, Ayuda en la entrega de software con mas rapidez, ademas de que da un mejor estándar dentro de las operaciones de las mismas aplicaciones, ademas de que es muy recomendada para los desarrolladores que se encuentran trabajando con la arquitectura de micro-servicios debido a que Docker ofrece estandarización.
